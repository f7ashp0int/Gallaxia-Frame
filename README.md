# Gallaxia Frame

Gallaxia Frame is a simple web app that lets you upload an image, preview it with a frame overlay, and download the composed result.

## Develop locally

Requirements: Node.js and npm.

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Notes

- Replace `public/placeholder.svg` and metadata values in `index.html` with your own branding as needed.
